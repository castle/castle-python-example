The Privacy API (added in castle 7.0) helps you honor GDPR/CCPA requests by letting you request or delete the data Castle holds for a given user.

Provide an identifier (e.g. an email) and an identifier_type (e.g. $email), then either request or delete that user's data.

A valid Castle API secret is required for these calls to succeed.
